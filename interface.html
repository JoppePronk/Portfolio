De Boekenzoeker

Visual Interface Design

De opdracht.

Ik kreeg de opdracht om een user interface te maken voor het zoeken van één of meerdere boeken uit de schoolbibliotheek, voor scholieren uit de onderbouw. Daarbij is het dus belangrijk dat je goed de doelgroep weet aan te spreken.

Periode: 2 maanden

Individueel

Moodboard.

Eerst maakte ik een moodboard. Ik onderzocht wat de doelgroep allemaal bezighoudt. Ik had het eerste idee om een soort fortnight interface te maken. Dus de felle kleuren en de cartoon stijl die de game onderscheidt van alle andere games.

Maar ik liep een beetje vast op dat idee. Toen bedacht ik wat ik zelf altijd als kind speelde. Dat was Super Mario Bros op de ds. Na onderzoek hebben gedaan wist ik wat ik wilde maken: een game die je kan spelen terwijl je naar boeken zoekt.

States.

Tijdens het ontwerpen van de schermen waren de states ook van belang. Wat zou er bijvoorbeeld gebeuren als een zoekopdracht niks oplevert. Of hoe ziet een laadscherm eruit? Ik wilde dit allemaal in dezelfde stijl maken als de rest van de interface.

Styleguide.

Om niet het overzicht kwijt te raken in alle verschillende stijlelementen heb ik een stijlgids gemaakt.

Doorlopen van de Boekenzoeker.

In dit filmpje zie je hoe je door gebruik van de pijlen door de interface heen kan lopen. Stapsgewijs ga je door de interface heen, om zo te filteren naar jouw ideale boeken. In het resultatenscherm kun je ook het filter aanpassen. Als je uiteindelijk je boekenlijst gemaakt hebt, kun je deze uitprinten en op zoek gaan naar de gekozen boeken op je lijst.  Je speelt een game terwijl je naar boeken zoekt.
